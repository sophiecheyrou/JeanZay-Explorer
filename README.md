# Jean Zay Explorer — Module 1

Base stable de la nouvelle application.

## Fonctions incluses

- navigation fiable entre Accueil, Explorer, Équipe et Classement ;
- espace organisateur séparé (`admin.html`) ;
- charte graphique responsive ;
- chronomètre pour le 31 août 2026, de 9 h à 11 h ;
- création d’équipe enregistrée localement ;
- carte Google My Maps et 18 lieux filtrables ;
- classement lu depuis Google Apps Script ;
- vérification du backend dans l’espace organisateur ;
- PWA installable et cache hors connexion.

## Code organisateur

Le code `1600` s’utilise uniquement dans `admin.html`.

## Mise en ligne

1. Conserver une copie de la version V2.1 actuelle.
2. Déposer tous les fichiers de ce dossier à la racine du nouveau dépôt ou de la branche de test.
3. Activer GitHub Pages sur la branche choisie.
4. Ouvrir `index.html#accueil`.
5. Tester les quatre onglets et `admin.html`.

## Limites volontaires du module 1

Le GPS, la prise de photo et la validation des missions ne sont pas encore activés. Ils seront ajoutés dans les modules suivants afin de préserver une base simple et fiable.
